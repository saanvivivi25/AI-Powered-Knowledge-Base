o	An AI-powered document management and question-answering system built using FastAPI, Streamlit, SQLite, SQLAlchemy, and Google Gemini AI. The application allows users to securely upload PDF, DOCX, and TXT documents, automatically extract and analyze their content, and ask questions based on the uploaded documents.
	
	Features
o	User Registration and Login with JWT Authentication 
o	Secure Password Hashing using Bcrypt 
o	Upload PDF, DOCX, and TXT Documents 
o	Automatic Text Extraction 
o	AI-Generated Document Summary 
o	Key Points and Important Topics Extraction 
o	Ask Questions About Uploaded Documents 
o	View Uploaded Documents and AI Analysis 
o	Delete Uploaded Documents 
o	Background AI Processing 
o	SQLite Database Integration 
o	Request Logging and Error Handling 
o	Interactive Streamlit Dashboard 
o	REST API with FastAPI 
o	Swagger API Documentation 
	
    Technologies Used
o	Python 
o	FastAPI 
o	Streamlit 
o	SQLAlchemy (Async) 
o	SQLite 
o	Google Gemini API 
o	LangChain 
o	JWT Authentication 
o	Pydantic 
o	Bcrypt 
o	PyPDF 
o	Python-Docx 
	
	Project Workflow
o	Register and log in securely. 
o	Upload a PDF, DOCX, or TXT document. 
o	Extract text from the uploaded document. 
o	Generate AI-powered summaries, key points, and topics. 
o	Store document information in SQLite. 
o	View uploaded documents and analysis. 
o	Ask AI questions related to any uploaded document. 
o	Delete documents when no longer needed. 
	
-	This project demonstrates authentication, asynchronous APIs, file processing, AI integration, background tasks, database management, logging, and modern web application development using FastAPI and Streamlit.
